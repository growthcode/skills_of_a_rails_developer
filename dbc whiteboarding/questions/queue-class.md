#Write a Queue Class

Implement a queue class. You must write all five methods listed below. 

### Stack and Queue Cheat Sheet

**Queue: First in, first out**
- first
- last
- enqueue(item) #add
- dequeue #remove
- is_empty?

**Stack: Last in, first out**
- peek
- pop #remove
- push(item) #add
- is_empty?
